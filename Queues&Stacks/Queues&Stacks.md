# Queues and Stacks :muscle:

**stacks** - first in last out (FILO)
**queues** - first in first out (FIFO)

## Stacks

| Method | Big O |
| ------ | ----- |
| lookup | O(n)  |
| pop    | O(1)  |
| push   | O(1)  |
| peek   | O(1)  |

## Ques

| Method  | Big O |
| ------- | ----- |
| lookup  | O(n)  |
| enqueue | O(1)  |
| dequeue | O(1)  |
| peek    | O(1)  |
